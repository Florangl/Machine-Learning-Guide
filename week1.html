Week 1 — Polynomial Regression
Fokus: kapan linear model gagal, kapan polynomial regression dibutuhkan, cara baca bentuk kurva, dan makna koefisien.
Konsep Inti

Polynomial Regression

Model regresi yang menambahkan X², X³, dan seterusnya agar hubungan yang melengkung tetap bisa dimodelkan. Model ini non-linear di X, tetapi tetap linear di parameter/koefisien.

⚡ Definisi wajib
Kapan Dipakai?

Saat Linear Tidak Cukup

Kalau scatter plot menunjukkan pola naik-lalu-melandai, U-shape, atau S-shape, maka garis lurus tidak cukup. Di slide week 1, contoh hubungan Electives → Interest jelas melengkung sehingga model linear bias.

Cek scatter plot dulu
Shape

Order Menentukan Lengkungan

Linear = 0 bend, Quadratic = 1 bend, Cubic = 2 bends. Rule cepat: jumlah lengkungan = order - 1.

⚡ Sering keluar
Koefisien

Tanda X² Menentukan Arah

Jika koefisien X² positif, kurva membuka ke atas (U). Jika negatif, membuka ke bawah (∩). Jadi tanda koefisien tertinggi menentukan arah curvature.

Wajib hafal
Centering

Mengubah Interpretasi

Centering X berarti menggeser titik acuan dari X=0 ke mean(X). Fit model sama, tapi interpretasi koefisien linear menjadi ‘efek pada nilai rata-rata X’.

⚡ Jebakan dosen
Calculus

Turunan = Perubahan Slope

Untuk model Y = a + bX + cX², maka slope lokal adalah dY/dX = b + 2cX. Jadi efek X terhadap Y tidak konstan; berubah di setiap titik X.

Analogi utama

Bayangkan kamu menuang air ke mangkuk. Garis lurus cocok untuk jalan datar, tapi mangkuk itu melengkung. Polynomial regression adalah cara menyesuaikan model dengan bentuk mangkuk, bukan memaksa semua jadi garis lurus.

🎯 Cara pakai analogi ini

  • Pakai analogi untuk ingat alurnya, bukan mengganti definisi formal.
  • Setelah paham analogi, cocokkan lagi ke istilah asli bahasa Inggris.
  • Kalau bisa menjelaskan dengan analogi + istilah formal, biasanya kamu sudah benar-benar paham.

⚠️ Jebakan Dosen
Kalimat atau konsep yang sering bikin mahasiswa salah.
⚠️ Polynomial regression bukan berarti modelnya tidak linear sama sekali. Ia tetap linear pada koefisien.
⚠️ Koefisien linear bukan efek global tetap; pada model quadratic, koefisien linear adalah slope di titik acuan.
⚠️ Model linear yang signifikan belum tentu bagus jika plot residual menunjukkan pola melengkung.
⚠️ Centering tidak mengubah kualitas fit model, tapi mengubah makna koefisien.
