# Machine Learning Study Guide

A personal study guide I built to help prepare for exams 
in the Machine Learning course at Institut Teknologi Del.

## What's inside
- Concept cards for each week's material
- Key terms explained in simple Indonesian
- Practice quizzes with instant feedback
- Exam tips and common mistakes to avoid

## Why I built this
Studying from slides alone wasn't enough, so I turned 
my notes into something more interactive and fun to use.

## Built with
HTML, CSS, and JavaScript — no libraries, no frameworks.
Just a single file that runs in any browser.
